#  **Job Scheduler \& Automation System**



This project is a mini automation system that allows users to create,

run, and monitor background jobs with webhook integration.

Built for Dotix Technologies skill test.



**Tech Stack**



Frontend: React, Tailwind  

Backend: Node.js, Express  

Database: SQLite  



**Architecture**



Frontend → Backend → Database → Webhook



 **Setup**



1\. cd backend

2\. npm install

3\. node index.js



4\. cd frontend

5\. npx serve .



**APIs**



POST /jobs  

GET /jobs  

POST /run-job/:id



**Features**



\- Create Job

\- Run Job

\- Filters

\- Webhook
